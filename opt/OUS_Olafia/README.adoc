== Oversettelse 

Anamnese_translations.xml :: er XML fil som inneholder oversettelse av kodeverk brukt i 

Anamnese v7-LOC.opt :: Er OPT som blir manuelt oppdatert med oversatte koder 

Engelsk oversettelse Selvregistreringsskjemaet Anne og Patricia.xlsx :: Er Excel arket som er master for oversettelsene 


WARNING: Siden arketypene skal brukes på flere språk BØR alle kodeverk legges over til arketyper. Da kan vi støtte den sanne multi-språklige naturen til arketyper og det er mulig med internasjonal review. Tenk f.eks. at en russisk talende kliniker kan bistå med oversettelsen til russisk - og disse arketypen tas rett inn i Norge.
